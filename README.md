# FinWise

A student-led extracurricular project exploring **UN Sustainable Development Goal 8: Decent Work and Economic Growth** through financial literacy education and interactive tools.

## 🌍 About

This website was built as part of a 12-week extracurricular summer project focused on SDG 8. The site connects global economic development to everyday personal finance — demonstrating how financial literacy is both a personal skill and a driver of broader economic empowerment.

## 📄 Pages

| Page | Description |
|------|-------------|
| **Home** | Hero section with SDG 8 context and global statistics |
| **Learn** | 4 educational modules: Budgeting, Saving, Debt, Income |
| **Tools** | Interactive budget calculator + savings goal tracker |
| **Global Impact** | Data, statistics, and SDG 8 targets framework |
| **About** | Project background and methodology |

## 🛠️ Features

- **Budget Calculator** — Enter monthly income and expenses; get a personalised breakdown with percentage bars and tips
- **Savings Goal Tracker** — Set a target amount and timeframe; calculate required monthly savings
- **Fully client-side** — No backend, no data stored, works from a single HTML file
- **Mobile responsive** — Works across all screen sizes
- **SPA navigation** — Smooth single-page routing without page reloads

## 🎨 Design

- **Palette:** UN SDG-inspired greens, blues, and warm neutrals
- **Typography:** Playfair Display (headings) + DM Sans (body)
- **Aesthetic:** Editorial warmth — professional but approachable

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR-USERNAME/sdg8-financial-literacy.git

# Open in browser — no build step required
open index.html
```

Or deploy instantly to:
- **GitHub Pages** — push to `main` branch, enable Pages in repo settings → select root
- **Netlify** — drag the folder into netlify.com/drop
- **Vercel** — `vercel deploy` from the project folder
